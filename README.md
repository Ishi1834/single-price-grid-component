### What I learned

How to use css Grid:

```css
main {
  display: grid;
  grid-template-columns: 1fr;
  margin: 15vw 6vw 5vw 6vw;
  max-width: 500px;
  border-radius: 10px;
  overflow: hidden;
  box-shadow: rgba(0, 0, 0, 0.1) 0px 10px 30px, rgba(0, 0, 0, 0.1) 0px 5px 60px;
}
```
